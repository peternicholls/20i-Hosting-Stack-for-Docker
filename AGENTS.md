# Agents

This repository supports OpenAI Codex–style agents (for example, GitHub Copilot).

All agent definitions and instructions are located in the `.github/agents/` directory:

- **copilot-instructions.md** — Base instructions for GitHub Copilot
- **speckit.analyze.agent.md** — Analysis agent
- **speckit.checklist.agent.md** — Checklist management agent
- **speckit.clarify.agent.md** — Clarification agent
- **speckit.constitution.agent.md** — Project constitution agent
- **speckit.implement.agent.md** — Implementation agent
- **speckit.plan.agent.md** — Planning agent
- **speckit.research.agent.md** — Research agent
- **speckit.specify.agent.md** — Specification agent
- **speckit.tasks.agent.md** — Task management agent
- **speckit.taskstoissues.agent.md** — Tasks-to-issues converter agent

These agents are automatically available when using GitHub Copilot in this repository.
